### To create a project:

```bash
php index.php -f project:Project
```
* For compound names
```bash
php index.php -f project:'My Project'
```

### To create a module:
```bash
php index.php -f module:MyProject:MyModule
```

### To create a scaffold:
```bash
php index.php -f skeleton:MyProject:MyModule
```
